# スーパービンゴトーナメント

**【STAGE１　スーパービンゴ】**  
・**1m9mの代わりに7p7sが入っている**（7p7sが8枚ずつ）  
・7は手牌で**4枚までしか使えない**  
・**東風戦**30000 点持ち 30000点返し、**0点はトビ**  
・完全着順制 一人浮き+60枚/-20枚　二人浮き＋60枚/0枚/-60枚  
・**オーラスの親の連荘は任意、**親はノーテンでも連荘  
・**ラスの人が親スタート**  
・**アリアリ、ツモ損無し、立直チューリップ**  
・**５s５p各2枚**、花牌4枚、**北は役牌**  
・フリテンリーチ、リーチ後の見逃しアリ  
・祝儀は赤牌、一発、裏ドラ、チューリップ  
・役満は20枚（ロンもツモも一緒）  
・七士無双アリ（7の暗槓ロンアリ）  
・**白ポッチはリーチ後永久オールマイティ**  
・**一発、白ポッチ、トリプル以上で転落回避1回**  
**【確変】**  
**7sを3枚以上または7pを3枚以上**使ってアガると**上下段チューリップ**。上下滑るまで継続する  
**【16R】**  
**7s7pをどちらも3枚以上**使ってアガると、**上下段チューリップ＋最終獲得チップが倍**になる  
**【突確】**  
**1枚のチューリップで6枚以上ヒット**すると**チューリップの状態がひとつ上がる**  
**【STAGE2～　満塁ホームラン.ver】**  
**〇満塁：**トんだ局に支払う祝儀が倍になる。  
**〇ホームラン：**アガった局に貰う祝儀が倍になる。  
・全員【満塁】でスタートする。  
・アガると【ホームラン】になる。  
・【ホームラン】は他者の和了による失点で解除され【満塁】に戻る。  
・【ホームラン】が【満塁】を飛ばすと祝儀が**４倍**となる。  
・【ホームラン】が【満塁】を２人トバしすると【サヨナラ】。祝儀が**６倍**。
